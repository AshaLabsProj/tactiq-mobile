# Tactiq Coach Visual Direction

## Product Character

Tactiq Coach should feel like **calm sideline intelligence** for youth football coaches. The product is not a dense analytics dashboard; it is a composed assistant coach that helps the user notice, record, and act on the next useful signal.

## Visual Language

The interface uses a chalk-white canvas, white cards, and an emerald-led action system. Deep pitch green provides decisive actions and selected states, while pale sage creates calm context panels. Tactical amber is reserved for focus areas, missing reviews, and coaching priorities. Blue is avoided except where it is inseparable from legacy source imagery.

Recurring motifs include pitch-zone grids, movement arrows, formation dots, field lines, and compact performance numerals. These motifs should remain functional and restrained rather than decorative.

## Typography and Hierarchy

Page titles are large, dark, and confident. Small uppercase labels identify team context, workflow steps, and signal categories. Stat numerals use tabular figures where possible. Supporting copy remains short and practical, with generous line height and no marketing language inside active workflows.

## Interaction Rules

Each screen prioritizes one primary action. Primary buttons use deep emerald and provide light haptic feedback. Lists and cards use opacity feedback. Match capture keeps the essential sequence visible: **Zone → Outcome → Record**. Assessment keeps the essential sequence visible: **Player → Skills → Save**.

## Copy Voice

Copy should sound like a composed assistant coach: short, match-aware, practical, and focused on the next decision. Prefer “Prepare for Northside Juniors” over generic labels such as “Upcoming event,” and “What to do next” over abstract analytics terminology.

## Mobile Constraints

Design for portrait orientation, one-handed use, 44-point minimum touch targets, readable contrast, and content within safe areas. Bottom navigation contains only Home, Squad, Capture, and Insights. Accounts and cloud controls are excluded from this local-first release unless explicitly requested later.

## Visual Verification Notes

Portrait previews confirmed that the four-tab hierarchy, card spacing, high-contrast emerald actions, and one-action-per-screen approach remain clear at 390 × 844 points. Detailed flow review also confirmed readable player profiles, match setup, live capture, and match summaries. Two small-screen issues were corrected: qualitative assessment labels now sit above rather than inside rating controls, and pitch/heat-map third labels now have dedicated columns instead of overlapping event cells.

The final launcher mark uses an off-white forward tactical path over a deep emerald pitch-centre motif. It remains legible at small scale, contains no text, fills the square, and uses no blue.
